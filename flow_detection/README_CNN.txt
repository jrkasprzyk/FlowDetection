For these, please use the following order: 1. cnn_AI_create -> 2. AI_test -> 3. analyze_photos_using_AI.

1. cnn_AI_create:
Through this program, you can create AI that detects flow. The output of this program is a ".h" file. To create this, you must place appropriate learning photos in the 'supervisor' file. Afterwards, just do what “need to type” or “need to change” asks in the code.

2.AI_test:
You can experiment with photos of your choice to see if the generated AI works well.

3. analyze_photos_using_AI:
You can analyze the photos you want. Enter the desired result path and check the program results.
